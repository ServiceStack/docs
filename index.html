Community Support #

Trial & Free License Users can access the community support options below:

Ask on ServiceStack/Discuss

Receive answers from ServiceStack community in the new ServiceStack/Discus Q&A Community Support category on GitHub Discussions.

Ask on StackOverflow

Receive answers from the .NET community on StackOverflow.

StackOverflow questions with the servicestack hash tag is frequently monitored by ServiceStack developers.

Commercial Support #

Registered developers with an active Commercial License can access technical support directly from ServiceStack engineers via the commercial support channels below:

Customer Support Forums

We're here to help! Active Customers can access technical support from our Customer Forums.

Developers emails needs to match an account's email or a registered support contact.

Report an issue

If you've found an issue in any of our libraries, please report it to our GitHub Issues repository.

Request a Feature

Request a new Feature or additional docs, samples and use-cases you would like to see in ServiceStack.

Premium Support

Need direct, private support from ServiceStack expert engineers?

Go to your Support page to access your premium support options.
